¿Qué es el Método de Jacobi?

En metodos numéricos el método de Jacobi es un método iterativo, usado para resolver sistemas de ecuaciones lineales del tipo A x = b , donde A es una matriz cuadrada de coeficientes, x es el vector columna de las incógnitas y b es el vector columna de términos independientes. El algoritmo toma su nombre del matemático alemán Carl Gustav Jakob Jacobi. El método de Jacobi consiste en usar fórmulas como iteración de punto fijo.
