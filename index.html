¿Qué es el Método de Jacobi?

En metodos numéricos el método de Jacobi es un método iterativo, usado para resolver sistemas de ecuaciones lineales del tipo A x = b , donde A es una matriz cuadrada de coeficientes, x es el vector columna de las incógnitas y b es el vector columna de términos independientes. El algoritmo toma su nombre del matemático alemán Carl Gustav Jakob Jacobi. El método de Jacobi consiste en usar fórmulas como iteración de punto fijo.

Algoritmo del Método de Jacobi

Pasos del Algoritmo:

  1. Verificar convergencia: La matriz A debe ser diagonalmente dominante
  2. Despejar variables: De cada ecuación despejamos una variable diferente
  3. Valores iniciales: Establecer valores iniciales para las variables
  4. Iteración: Aplicar las fórmulas iterativas hasta convergencia
  5. Criterio de parada: |xi(k+1) - xi(k)| < ε

Fórmula General:

x (k+1) = bi - ∑ j=1,j≠i n aij x j (k) aii

Ventajas y Desventajas del Método de Jacobi

✔ Ventajas

  • Fácil de implementar y entender
  • Paralelizable para sistemas grandes
  • Requiere poca memoria adicional
  • Estable numéricamente
  • No modifica la matriz original
  • Es muy eficiente para matrices dispersas

❌ Desventajas

  • Convergencia lenta comparado con Gauss-Seidel
  • Requiere que la matriz sea diagonalmente dominante
  • No siempre converge
  • Necesita muchas iteraciones para alta precisión
  • Sensible a la elección de valores iniciales
